# Instalation Instructions

### Installation 
#### Clone This Projects 

```bash
git clone git@github.com:hananloser/take_home_project_books_lib.git

```

#### Install Dependencies
```bash
yarn install
```

## In Production Mode
```bash
yarn build && node dist/main.js
```

## In development Mode
```
yarn dev
```

## Running Test
```
yarn test
```
